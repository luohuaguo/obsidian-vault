# 「晒过程·客户见证/产品推广型」爆款文案框架

> 数据支撑：9条S/A级爆款 | 平均点赞12,822

## 一、核心选题公式

公式：客户身份标签 × 遇到的具体卡点 × 转变/突破瞬间

| 变量         | 可选值示例                                                   |
|-------------|-------------------------------------------------------------|
| 客户身份标签 | 律师/护肤老师/教育博主/电商主播/地产一姐/产后康复师/阅读博主     |
| 具体卡点     | 有流量没转化 / 线下强线上弱 / 学了用不上 / 内容好但不火 / 方法不适合自己 |
| 转变瞬间     | 捅破窗户纸 / 终于知道正道怎么走 / 回去就能调整 / 一下就有信心了     |

底层情绪驱动力：
- 希望（他能做到我也能）
- 焦虑缓解（原来问题出在这里）
- 信任建立（真实客户的真实反馈）

## 二、开篇设计

优先级排序：对话式 > 痛点提问式

### 方向A：悬念引入

句式模板：
- 「怎么就有这么精准的客户？我们一直认为这里面一定有个【比喻：窗户纸/密码/关键点】没有【动作：捅破/找到/打通】。」
- 「我来之前我觉得【旧认知】。没想到能细到这种程度。」

### 方向B：身份+困境

句式模板：
- 「过了【时间】没有什么结果，人很崩溃。【赛道】一直被【困境】。上完课程以后呢，就一下知道我的问题出在哪里了。」
- 「我之前在【城市】的时候，基本上都是靠【旧方法】，对【新事物】还是比较犹豫的。」

### 方向C：情感触发

句式模板：
- 「同行的人像礼物一样。好久不见，你跑到我这儿来【目的】，来找找【感觉】，【感觉】是啥？」

## 三、论证过程设计

推荐结构：「困境→转折→方法→结果」四段式（3个论点段）

### 骨架：

论点1（占全文约30%）：
→ 客户原来的做法和困境
→ 描述具体场景和痛苦感受
→ 要有情绪词（焦虑/崩溃/不知道怎么办/极其焦虑）
→ 小结论：以前的路走不通

论点2（占全文约35%）：
→ 转折点——发现了什么/学到了什么
→ 用客户自己的话说出恍然大悟的感觉
→ 具体方法点到为止，不展开（制造好奇）
→ 小结论：原来正道是这样的

论点3（占全文约20%）：
→ 现在的做法和变化
→ 案例论证（具体的改变和效果）
→ 回扣主题

### 关键技巧：
- 让客户用自己的话说出问题和收获，而非IP本人总结
- 真实感是该类型的命脉——口语化、有犹豫、有情绪波动
- 方法论只点到不讲透（制造"我也想学"的欲望）
- 必须有具体的"前后对比"（之前XX → 现在XX）

### 可选论据方向：

| 论据方向       | 典型用法                                                   |
|---------------|-----------------------------------------------------------|
| 前后对比       | 「以前写文案脑袋里没东西 → 现在知道每一条给谁看」               |
| 恍然大悟时刻   | 「我一下就知道那个正道怎么走了，窗户纸捅破了」                  |
| 具体方法引子   | 「比如说认知型的视频怎么说、成交型的视频怎么做」（只列不展开）    |
| 情感真实流露   | 「极其焦虑 / 我不搞了 / 带着巨大的压力感」                     |

## 四、结尾强化

首选：观点收束式 / 总结归纳式

句式模板A（客户自己说出行动计划）：
- 「所以今天我梳理完之后，回去又得调整内容。」
- 「我脑袋都有概念了，我学到了有价值的事情，我都会在内部做转型。」

句式模板B（情感真实流露）：
- 「我非常非常感谢【IP名】，你真的让我突然有了信心，我会觉得我可以做好这件事情。」
- 「真的让我能量满满。」

句式模板C（引流转化式，仅用于产品推广目的）：
- 「这套方法叫【方法名】，包含【要素1】、【要素2】、【要素3】。在评论区打【关键词】，看这个图领取。」

## 五、选题生成技巧

快速批量生成法：

第一步：列出已服务过的不同行业客户
第二步：每个客户提取一个「转变前后」的对比
第三步：包装为客户视角的故事

示例：
- 律师客户 × 从追求流量到追求转化 = 「律师做线上获客，必经的三个阶段」
- 护肤老师 × 从线下强到线上也强 = 「专家IP获客，从用户角度了解她们要什么」
- 教育博主 × 从没结果到知道怎么复盘 = 「好好复盘，直播没有想象中那么难」
