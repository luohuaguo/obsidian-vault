# 「聊观点·行业导向型」爆款文案框架

> 数据支撑：6条S/A级爆款 | 平均点赞10,447 | 平均分享1,486

## 一、核心选题公式

公式：行业里「人人都做但没人敢说」的事 × 犀利态度

| 变量                 | 可选值示例                                              |
|---------------------|--------------------------------------------------------|
| 人人都做但没人敢说的事 | 社交就是功利的 / 拒绝客户是为了赚更多 / 输入不够导致输出不行 / 复盘只看数据没有用 |
| 犀利态度             | 直说不装 / 承认现实比假装高尚强 / 批判常见做法 / 撕掉遮羞布   |

底层情绪驱动力：
- 认同（终于有人把我想说的说出来了）
- 争议（我不同意，我要说两句）→ 直接拉评论和分享
- 爽感（痛快、解气、过瘾）


## 二、开篇设计

优先级排序：警告式 = 对话式（均有效）

### 方向A：直接亮观点（制造认知冲突）

句式模板：
- 「【城市/行业/圈子】的氛围不好？【城市/行业/圈子】的氛围是现实。」
- 「我真的不希望大家误解。我觉得我一直说我【争议行为】是为了【表面原因】，我不是这个目的。」
- 「【争议性判断】，这在当下是非常重要的。」

### 方向B：先承认再翻转

句式模板：
- 「今年我们到现在为止，拒绝的客户两三百万应该是有的。」（先抛事实，再解释逻辑）
- 「我现在越来越觉得，如果你想做好【事业】，你就要长期保持【某个反直觉的习惯】。」

### 方向C：洞察式切入

句式模板：
- 「网上不是有一个很火的段子嘛——【段子内容】。因为【深层原因】。」
- 「之前从来没有这样【负面结果】过，从【高点】掉到【低点】。所以你现在能不能估算出来是哪些原因？」


## 三、论证过程设计

推荐结构：递进式（3-4个论点层层加码）

### 骨架：

论点1（占全文约20%）：
→ 现象描述——大家都在做什么/都在想什么
→ 制造共鸣（你是不是也这样）
→ 小结论：表面看是这样

论点2（占全文约30%）：
→ 翻转——为什么这么做其实是对的/错的
→ 给出你的犀利判断
→ 用具体案例佐证
→ 小结论：真相是这样的

论点3（占全文约25%）：
→ 更深一层——这背后的本质是什么
→ 升华到更大的逻辑（商业本质/人性规律/行业趋势）
→ 案例论证+回扣主题

（可选）论点4（占全文约15%）：
→ 回到个人——我是怎么做的
→ 用自身案例落地

### 论据组合原则：
- 必须有「现象→本质」的层层递进
- 每一层都要比上一层更深/更尖锐
- 可以用极端案例制造冲击力
- 态度鲜明，不要两头讨好

### 可选论据方向：

| 论据方向   | 典型用法                                                    |
|-----------|----------------------------------------------------------- |
| 现实案例   | 「他判断你跟他不是同层级的时候，你给他发消息不回，你主动加他不通过」 |
| 反转逻辑   | 「你说这个事不好吗？有的人觉得功利，我觉得这就是能发展好的原因」   |
| 因果推理   | 「信息流动快 → 财富流动就快」                                  |
| 自身经历   | 「今年我们拒绝的客户两三百万应该有的」                          |


## 四、结尾强化

首选：观点收束式（留白=留争议=留评论——该类型平均评论291）

句式模板：
- 「【城市/行业】为什么【结果】？不就是因为这点。」
- 「你不要自己去骗自己，你关注那个事情到底怎么被解决，好不好？」
- 「人家公司【大数字】员工，他都可以做到【反直觉做法】，你凭什么做不到？你如果做不到，就是你【本质问题】。」

注意：该类型的结尾绝对不要"总结归纳"，越留白越有争议，越有争议越有评论和分享。


## 五、选题生成技巧

快速批量生成法：

第一步：列出你所在行业里的「政治正确说法」
第二步：逐一反问——这个说法是真的吗？现实是什么？
第三步：找到「政治正确≠现实真相」的那几个
第四步：用犀利态度重新表达

示例：
- 政治正确「人脉靠真诚」→ 现实「成年人世界就是这么现实」
- 政治正确「拒绝客户是有良心」→ 现实「拒绝客户是为了更长久获利」
- 政治正确「要持续输出」→ 现实「不输入就不可能持续输出」
- 政治正确「复盘很重要」→ 现实「你的复盘方法根本就是错的」
