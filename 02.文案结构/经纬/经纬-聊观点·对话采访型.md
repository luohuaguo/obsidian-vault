# 「聊观点·对话采访型」爆款文案框架

> 数据支撑：11条S/A级爆款 | 平均点赞14,767 | 平均评论1,017 | 平均分享1,883
> 综合数据全类型最强，唯一能同时拉高点赞+评论+分享三项指标的类型

## 一、核心选题公式

公式：知名人物/头部案例 × 被忽略的维度 × 颠覆性结论

| 变量         | 可选值示例                                                         |
|--------------|-------------------------------------------------------------------|
| 知名人物/案例 | 雷军、李想、董太太、杨璇、罗永浩、行业头部同行、热门事件当事人         |
| 被忽略的维度  | 营销手法vs做事态度 / 表面专业vs客户视角 / 流量思维vs转化思维 / 公关能力vs产品能力 |
| 颠覆性结论    | 比起A我更喜欢B / A的成功不是因为你以为的X而是Y / 你学不会是因为你只看到了表面 |

底层情绪驱动力：
- 好奇（我也想知道真相）
- 认同（我也这么想但说不清楚）
- 争议（我不同意，我要评论）

## 二、开篇设计

优先级排序：警告式 > 对话式 > 痛点提问式

### 方向A：制造对立（数据最佳，平均分享1,672）

句式模板：
- 「我真的对【对象A】大为改观。你可是【数字】个拥护【对象B】的，我是认【对象B】的【某项能力】。」
- 「【权威人物】当年在【场景】跟所有人说，不要跟【对象】比【能力】。」

### 方向B：经历引入

句式模板：
- 「我最近【时间段】干了一个事情，我把【信息源】全部都【动作】了。我只能这么跟你讲，【强烈感受词】。」
- 「我挑几个对我影响非常大的【认知/启发】来跟你分享。」

### 方向C：数字震撼（平均点赞17,563）

句式模板：
- 「从【起点数字/状态】到【终点数字/状态】，这条视频给你讲清楚【核心问题】。」
- 「你最大的【指标】多少？【具体数字】。但是呢【转折——来的人/结果都不对】。」


## 三、论证过程设计

推荐结构：多案例并列对比 + 逐层递进（3个论点段）

### 骨架：

论点1（占全文约25%）：
→ 先讲A的做法
→ 用具体案例/数据佐证
→ 分析为什么有效
→ 得出小结论

论点2（占全文约30%）：
→ 再讲B的做法
→ 与A形成对比，揭示差异
→ 用具体案例佐证
→ 得出小结论

论点3（占全文约20%）：
→ 提炼差异背后的底层逻辑
→ 用案例论证
→ 回扣开篇抛出的核心问题

### 论据组合原则：
- 至少2个具体案例，且来自不同维度
- 必须有对比（A vs B / 过去 vs 现在 / 表象 vs 本质）
- 最后一个论点必须回扣开篇核心问题
- 案例要有细节（具体的话、具体的场景、具体的数字）

### 可选论据方向：

| 论据方向   | 典型用法                                              |
|-----------|------------------------------------------------------|
| 对比反差   | 「雷军这么说 vs 李想那么说」                             |
| 具体案例   | 「他当时怎么回答的？他说……」                             |
| 类比迁移   | 「消费者跟商家之间，其实他永远是一种博弈的关系」             |
| 反问连击   | 「你觉得劝得动他吗？你能改变她吗？你改变得了他吗？」        |
| 数据佐证   | 「小米SU7卖了30万辆」「公司3万名员工，朝九晚六」           |

## 四、结尾强化

首选：观点收束式（平均评论486，平均分享1,202——留白=留争议=留评论）

句式模板：
- 「我最后为什么会被【对象】打动？是因为我被他【具体经历】之后所流露出的【深层价值观】所打动了。我觉得这就是【升华概念】独特的魅力。」
- 「所以很多东西你模仿不来，因为你不知道为什么，只知道怎么做，就是走死路。」
- 「以前你不懂的时候，你觉得他好像【表面判断】。其实【真相】，只是视角，你之所以没看到。」

备选：总结归纳式（平均点赞13,396——观点明确，易获认同）

句式模板：
- 「他只是把他【线下能力】结合【线上特性】综合起来，达到了一个爆炸性的效果。」

## 五、选题生成技巧

快速批量生成法：

第一步：选一个行业热门人物/事件
第二步：切换2-3个分析维度（营销/产品/管理/价值观/用户关系）
第三步：与另一个人物/事件做对比
→ 1个人物可生成6-9个选题

示例：
- 雷军 × 营销手法 vs 李想 × 做事态度 = 「比起雷军，我更喜欢李想」
- 董太太 × 人设设计 vs 普通IP × 常规做法 = 「董太太聪明？我看没那么简单」
- 杨璇 × 内容策略 vs 同行 × 专业硬砸 = 「他看起来不讲专业，其实全是专业」
