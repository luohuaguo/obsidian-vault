# 「讲故事·小有成就型（讲自己）」爆款文案框架

> 数据支撑：6条S/A级爆款 | 平均点赞6,426
> 适合做人设建设和粉丝粘性，非引爆型内容但稳定出量


## 一、核心选题公式

公式：自身经历中的「反差时刻」× 提炼出的认知升级

| 变量       | 可选值示例                                                      |
|-----------|----------------------------------------------------------------|
| 反差时刻   | 从打工仔到天花板 / 执行力拉满的一年 / 别人吐槽我秀案例 / 1500万播放但来的人都不对 |
| 认知升级   | 执行力=顶级认知 / 有洞察比迎合重要 / 内容必须结合产品 / 流量大≠客资多   |

底层情绪驱动力：
- 激励（他能做到我也能）
- 好奇（他具体怎么做的）
- 信任感（真实、不包装、有血有肉）


## 二、开篇设计

优先级排序：对话式 > 警告式

### 方向A：数字冲击+态度宣言

句式模板：
- 「执行力拉满就是顶级认知。现在的情况是哪都没有流量红利了，所有行业都得卷的要死，你唯一能做的就是【核心主张】。」
- 「不说远的，【过去时间】我还在【起点身份】，【现在时间】我们现在算是【成就描述】了吧，我们【时间段】干了别人【更长时间】干不出来的【成果】。」

### 方向B：自嘲/真诚引入

句式模板：
- 「我有时候很羡慕那种能够讲空话的人。我们自己做了这么多年，还是会觉得每一条视频很难。」
- 「很多人吐槽我天天秀案例。如果你一年也能做这么多，麻烦你赶紧秀。」

### 方向C：具体数字+反转

句式模板：
- 「你最大的流量多少？【大数字】单条视频。但是呢来的人都不对。」
- 「讲干货对于成交的效果不大，本身并不能塑造很强的信任感。什么东西能够拉近信任距离？」


## 三、论证过程设计

推荐结构：时间线叙事（3个论点段）

### 骨架：

论点1（占全文约25%）：
→ 过去——我曾经的状态/做法
→ 要真实不包装（踩过的坑、走过的弯路、犯过的错）
→ 具体案例/数据佐证
→ 小结论：那时候我以为XX

论点2（占全文约35%）：
→ 转折——发生了什么让我改变
→ 具体的事件/人物/数据
→ 描述「恍然大悟」或「被现实打脸」的瞬间
→ 小结论：我才明白XX

论点3（占全文约20%）：
→ 现在——我的做法和理解
→ 用案例论证新认知
→ 回扣主题

### 论据组合原则：
- 必须有具体的时间节点（2022年/去年/那个月）
- 必须有真实的数字（1500万播放/两三百万/几十人在线）
- 态度要鲜明——不要中庸，要有立场
- 可以适度自嘲，但落点一定是成长和认知提升

### 可选论据方向：

| 论据方向   | 典型用法                                                    |
|-----------|----------------------------------------------------------- |
| 时间对比   | 「2022年还是打工仔 → 2023年出来做抖音 → 现在天花板」            |
| 数据反差   | 「1500万播放，来的人都不对」                                   |
| 自我复盘   | 「比起迎合谁说话，我更倾向有自己的洞察」                         |
| 他人评价   | 「很多人吐槽我天天秀案例」→ 正面回应                            |
| 具体经历   | 「系统误判封号 → 自己找原因 → 最终解决」                        |


## 四、结尾强化

首选：观点收束式

句式模板：
- 回到开头的反差，给出自己沉淀后的态度/结论
- 「所以我有理由相信【结论】，他最起码是【核心品质】的。」
- 「你不要总想着是不是被【外部原因】了，还是好好自己去找原因。」
- 「我喜欢和优秀的人做朋友。」

备选：引流转化式（当故事服务于产品推广时）

句式模板：
- 「我们能够做到【成果】，靠的就是【方法名】。想了解的在评论区打【关键词】。」


## 五、选题生成技巧

快速批量生成法：

第一步：回忆自己过去1-3年里「认知被刷新」的5个瞬间
第二步：每个瞬间提炼出「以前以为XX → 后来发现XX」
第三步：包装为有态度的标题

示例：
- 以前以为流量大就好 → 后来发现精准比流量重要 = 「内容不结合产品，你流量再大也没客资」
- 以前以为要迎合观众 → 后来发现要有自己的洞察 = 「比起迎合谁说话，我更倾向有自己的洞察」
- 以前以为案例说明一切 → 后来发现案例需要方法论 = 「有人说我案例谁来做都能做，认真回应一下」
