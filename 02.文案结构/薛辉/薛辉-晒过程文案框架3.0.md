# 晒过程文案框架

## 一、晒过程的4个内容方向

### 1. 过程展示
- 进货筹备·咨询成交
- 技术服务·售后服务

### 2. 测评产品
- 对比测评
- 极限测评
- 化验测评

### 3. 任务挑战
- 高/低成本
- 限时间/地点

### 4. 事件体验
- 新奇体验
- 奇葩规则
- 角色互换/扮演

## 二、火车节模型

### 基本结构

关键步骤起因(起因一定要加钩子) → 关键步骤经过 → 关键步骤经过 → 关键步骤结果

转场 → 转场 → 转场

### 创作流程(8步法)

#### 第1步：确认晒什么
明确要展示的核心内容

#### 第2步：预描场景
预先构思整体场景框架

#### 第3步：联想画面
联想具体的视觉画面

#### 第4步：置身事外
以旁观者视角审视内容

#### 第5步：聚焦环节
找出最关键的几个环节

#### 第6步：提炼火车节
提炼出起因-经过-结果的节奏

#### 第7步：补充事件细节
丰富关键步骤的细节描述

#### 第8步：展示专业点
加入专业性内容增强可信度

### 补充细节要点

补充关键步骤中可能出现的细节：
- 手艺
- 技巧
- 原料
- 成品

## 三、建立期待感的6个技巧

### 1. [送温暖]
- 给养老院送礼物·给孤儿送礼物
- 大量买下弱势群体售卖的商品

### 2. [金钱]
- 花大钱·花小钱办大事
- 薅羊毛·赚钱·省钱

### 3. [盲盒]
- 抽到什么穿什么/吃什么
- 抽到什么做什么/用什么
- 抽到什么干什么/惊喜礼物

### 4. [对抗]
- 下赌注·吵架了·被阻止了
- 发生意外·故意整蛊

### 5. [验证/揭秘]
- 解秘真假·挑战权威
- 揭秘隐私·是否成功·刨根问底
- 验证成分·验证价格·解秘传言

### 6. [荷尔蒙]
- 黑丝·长腿·大胸·明星脸
- 腹肌·搭讪·变装·跳舞
- 诱惑·大尺度照片

## 四、晒过程脚本创作路径

### 完整创作流程

进店理由 → 4个方向 → 选题 → 火车节模型 → 加钩子 → 脚本自检

### 第一步：进店理由选择

从以下维度选择：
- 服务好
- 老板好
- 质量好
- 价格好
- 效果好
- 专业强
- 好评多
- 生意好
- 有特色
- 规模大
- 选择多
- 案例多
- 颜值高
- 性价比
- 便利性
- 有面子

注意：尽量选择两个维度以上进行拍摄策划

### 第二步：选择内容方向

从4个方向中选择：
- 过程展示
- 测评产品
- 任务挑战
- 事件体验

### 第三步：选题要点

- 确定具体选题
- 结合爆款元素(成本的、人群的、奇葩的、最差的、反差的、怀旧的、荷尔蒙的、头牌的)

### 第四步：火车节模型应用

起因(加钩子) → 经过(转场) → 经过(转场) ...... → 结果

### 第五步：加钩子原则

- 开头一定要加：起因部分必须有吸引力
- 经过少加：过程中适度使用钩子

### 第六步：脚本自检要点

#### 核心检查项

1. 进店理由检查
   - 说xxx好就主要体现什么好
   - 保持焦点集中，不要分散

1. 起因钩子检查
   - 起因有没有钩子
   - 钩子是否足够吸引人

1. 角色定位检查
   - 是质检员还是销售员
   - 保持客观中立或适度营销

1. 钩子分配检查
   - 几个经过给一个钩子
   - 避免过度使用或完全没有

## 五、进店理由策划表(实体店专用)

### 三大维度分类

#### 产品维度
- 效果好：通过使用产品/技术手艺等服务，能得到明显的效果
- 质量好：产品耐用、方便使用，用料扎实、安全可靠
- 专业强：从业时间长、获得奖项、熟练

#### 店铺维度
- 好评多：产品使用/服务后体验感好，客户夸赞、复购多、转介绍
- 服务好：服务热情、耐心贴心，售后负责，响应迅速、送货上门、仔细
- 生意好：顾客多、订单多，断货、排队
- 有特色：环境、菜品、服务和同行有区别
- 规模大：面积大、设备全、员工多，连锁店、展示实力、建立信任
- 选择多：产品种类多，好搭配，一站式服务
- 案例多：成功案例多，说服力强，增强信任度
- 颜值高：拍照好看、网红打卡，有装饰性、环境高档

#### 人的维度
- 性价比：同价位产品里质量好，同款产品里价格低/量更大、赠品多
- 老板好：老板实在、性格好，大方、豪爽洒脱、有爱心、没架子
- 便利性：使用方便、距离近，省时、省力
- 有面子：彰显品位，限量的，排队难排的，贵的、不好买的、牌子的、小众的

### 策划原则

尽量选择两个维度以上进行拍摄策划
