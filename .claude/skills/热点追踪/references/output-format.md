# 热点选题分析输出格式规范

## 目录
1. 整体结构
2. 格式要求
3. 示例模板

## 概览
本规范定义了热点选题分析的输出格式，确保输出内容结构清晰、信息完整、便于阅读和理解。

## 核心内容

### 整体结构

```
# 【赛道】热点选题分析报告
**分析时间**：YYYY-MM-DD HH:MM:SS
**分析范围**：最近3天
**热点数量**：N条

---

**🔥 包含的爆款元素**：
- [元素名称]：[匹配度和简要说明]
- [元素名称]：[匹配度和简要说明]

```

### 格式要求

#### 标题层级
- 使用 H1 (`# `) 作为主标题
- 使用 H2 (`## `) 作为章节标题
- 使用 H3 (`### `) 作为小节标题

#### 排名标识
- 使用 `第X名` 明确标注排名
- 使用emoji增强视觉效果：
  - 📊 表示排行榜
  - 🔥 表示爆款元素
  - 💡 表示选题方向
  - 📝 表示内容简介
  - 🔗 表示链接
  - 📈 表示总结和建议

#### 概率标注
- 格式：`XX%（高/中/低）`
- 高概率：≥70%
- 中概率：40-69%
- 低概率：<40%

#### 爆款元素列表
- 使用无序列表 (`- `) 列出每个元素
- 每个元素包含：元素名称 + 匹配度 + 简要说明
- 匹配度标注：高/中/低

#### 选题方向列表
- 使用数字列表 (`1. `) 列出每个选题方向
- 每个方向包含：方向名称 + 具体说明（切入点、目标受众、表达形式）

#### 链接格式
- 使用Markdown链接格式：`[链接文本](URL)`
- 确保URL完整且可访问

#### 时间格式
- 统一使用 ISO 格式：`YYYY-MM-DD HH:MM:SS`
- 或使用简体中文格式：`2026年2月13日`

## 示例模板

### 完整示例

```markdown
# 【科技】热点选题分析报告
**分析时间**：2026-02-13 14:30:00
**分析范围**：最近3天
**热点数量**：10条

---

## 📊 热点爆款排行榜

### 第1名 - AI大模型突破：某公司发布新一代智能助手，性能提升50%
**爆款概率**：75%（高）
**发布时间**：2026-02-13
**来源**：科技媒体

**🔥 包含的爆款元素**：
- 时效性：高（刚刚发布，信息新鲜）
- 创新性：高（技术突破，性能大幅提升）
- 实用性：中（潜在应用广泛，但需等待实际效果）
- 话题性：高（AI是当前热门话题，引发讨论）

**💡 爆款选题方向**：
1. 技术深度解析：从性能提升50%这个关键数据切入，深入分析技术创新点，目标受众为科技爱好者，使用技术文章形式
2. 行业影响分析：探讨对AI行业的影响和机遇，目标受众为行业从业者，使用分析报告形式
3. 未来趋势预测：基于此次突破预测AI发展趋势，目标受众为普通读者，使用科普文章形式

**📝 热点简介**：某科技巨头今日发布新一代AI智能助手，声称性能较上一代提升50%，在多项基准测试中超越竞争对手，引发行业广泛关注和讨论。

**🔗 原文链接**：https://example.com/news/ai-breakthrough

---

### 第2名 - 某公司宣布裁员1000人，业务重心转移至新领域
**爆款概率**：65%（中）
**发布时间**：2026-02-12
**来源**：财经媒体

**🔥 包含的爆款元素**：
- 时效性：高（刚刚宣布）
- 影响力：高（大公司裁员事件，影响广泛）
- 争议性：中（裁员引发争议，但也有合理性）
- 反转性：中（业务转移可能带来新机遇）

**💡 爆款选题方向**：
1. 深度调查报道：从裁员事件背后挖掘公司战略调整原因，目标受众为投资者和行业观察者，使用深度报道形式
2. 员工故事：关注受影响员工的经历和感受，目标受众为大众读者，使用人物故事形式
3. 行业趋势分析：分析该行业整体趋势和未来走向，目标受众为行业从业者，使用行业分析形式

**📝 热点简介**：某知名科技公司宣布裁员1000人，并将业务重心转向新兴领域，引发业内对行业发展趋势的热烈讨论。

**🔗 原文链接**：https://example.com/news/layoff-announcement

---

### 第3名 - 专家预测：某技术将在未来5年内改变行业格局
**爆款概率**：60%（中）
**发布时间**：2026-02-11
**来源**：行业媒体

**🔥 包含的爆款元素**：
- 预测性：高（对未来趋势的预判）
- 影响力：中（影响行业发展）
- 实用性：中（为从业者提供参考）
- 话题性：中（引发行业讨论）

**💡 爆款选题方向**：
1. 趋势解读：详细解读该技术的发展路径和应用前景，目标受众为行业从业者，使用趋势分析形式
2. 专家访谈：采访多位行业专家获取不同观点，目标受众为决策者，使用访谈形式
3. 投资机会分析：分析该技术带来的投资机会，目标受众为投资者，使用投资分析形式

**📝 热点简介**：多位行业专家预测，某新兴技术将在未来5年内彻底改变相关行业的格局，多家企业已开始布局该领域。

**🔗 原文链接**：https://example.com/news/tech-prediction

---

## 📈 总结与建议

### 高潜力热点汇总
1. AI大模型突破（爆款概率75%）
2. 某公司宣布裁员（爆款概率65%）
3. 专家预测某技术改变行业（爆款概率60%）

### 选题策略建议
1. 优先选择高概率热点，这些热点具有更强的传播潜力
2. 结合热点爆发元素，选择最符合目标受众需求的角度
3. 注意热点时效性，快速响应并产出内容

### 注意事项
1. 避免涉及敏感或争议性过大的内容
2. 确保选题方向与账号定位匹配
3. 保持内容质量和原创性，避免同质化
```

## 注意事项

### 内容完整性
- 每个热点必须包含：标题、爆款概率、发布时间、来源、爆款元素列表、选题方向、简介、链接
- 爆款元素至少列出匹配度高的2-3个
- 选题方向至少提供2-3个具体可行的建议

### 格式一致性
- 统一使用Markdown格式
- 统一使用emoji标识
- 统一时间格式
- 统一概率标注格式

### 可读性
- 使用清晰的层级结构
- 合理使用emoji增强视觉效果
- 确保内容简洁明了，避免冗长
- 使用列表和分段提高可读性

### 可操作性
- 选题方向必须具体可行，不能空泛
- 明确切入点、目标受众和表达形式
- 提供实际可执行的建议

## 数据来源标注

- 在热点简介中明确说明信息来源
- 原文链接必须完整且可访问
- 如有多个来源，列出主要来源
