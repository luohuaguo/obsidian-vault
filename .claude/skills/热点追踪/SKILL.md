# 热点选题分析

## 任务目标
- 本 Skill 用于：基于用户输入的赛道，智能分析最近三天的热点事件，预测爆款潜力和推荐选题方向
- 能力包含：联网搜索热点、爆款元素分析、概率预测、选题建议、智能排序
- 触发条件：用户表达"分析某某赛道的热点"、"推荐爆款选题"、"预测热点爆款概率"等意图

## 前置准备
- 依赖说明：scripts脚本所需的依赖包及版本
  ```
  requests>=2.28.0
  ```
- 非标准文件/文件夹准备：无

## 操作步骤

### 标准流程

1. **引导用户输入赛道**
   - 询问用户：您想分析哪个赛道的热点？（例如：科技、财经、教育、美妆等）
   - 确认赛道关键词的准确性和范围
   - 说明分析范围：最近3天内的热点事件

2. **调用脚本搜索热点**
   - 执行 `python scripts/search_hotspots.py --keyword "<赛道关键词>" --days 3 --count 10`
   - 获取脚本返回的热点列表（包含标题、简介、发布时间、来源、链接）
   - 验证搜索结果的质量和数量

3. **分析每个热点的爆款元素**（强制要求，严格执行）
   - **必须**严格按照 [references/爆款元素整理汇总.pdf](references/爆款元素整理汇总.pdf) 中定义的13大爆款元素进行分析
   - **13大爆款元素列表（必须完全一致）**：
     1. 成本元素/金钱相关
     2. 最差元素
     3. 荷尔蒙元素
     4. 人群元素/与你有关
     5. 场景元素
     6. 数据元素/数字敏感
     7. 对比/对立元素
     8. 反向操作/反认知元素
     9. 怀旧元素
     10. 奇葩/猎奇/揭秘元素
     11. 头牌元素
     12. 高维词汇/高维认知元素
     13. 钩子问题元素
   - **禁止**使用PDF中未定义的任何元素名称，例如：时效性、创新性、实用性、影响力、话题性、趋势性等
   - 对每个热点，逐一评估上述13个元素：
     - 判断该热点是否包含某个元素（高/中/低匹配度）
     - 仅列出匹配度为"高"或"中"的元素
     - 为每个匹配的元素提供PDF文档中定义的理由（参考定义、关键词、案例）
   - **必须**确保使用的元素名称与PDF文档完全一致，包括"/"符号
   - 示例格式：
     - 成本元素/金钱相关：高（强调低成本投入获得高回报）
     - 对比/对立元素：中（通过对比制造认知冲突）

4. **预测爆款概率**（强制要求）
   - **必须**基于匹配的爆款元素进行量化评估
   - 评分方法：
     - 每个高匹配度的元素记2分
     - 每个中匹配度的元素记1分
     - 元素组合效应：如果包含3个及以上高匹配度元素，额外加3分
   - 总分计算：Σ(元素得分) + 组合效应分
   - 爆款概率等级：
     - 高概率（≥70%）：总分≥10分
     - 中概率（40-69%）：总分6-9分
     - 低概率（<40%）：总分≤5分
   - **禁止**主观随意调整概率，必须基于量化评估

5. **生成选题方向**（强制要求）
   - **必须**基于匹配的爆款元素特点生成选题方向
   - 每个选题方向必须明确关联到至少1个爆款元素
   - 每个选题方向必须包含：
     - 核心角度/切入点（具体、可执行，避免空泛）
     - 关联的爆款元素名称（必须与13大元素列表完全一致，包括"/"符号）
     - 目标受众（明确的用户群体）
     - 表达形式建议（文章/视频/图文/直播等）
   - 选题方向必须具有可操作性和爆款潜力
   - 示例格式：
     1. 深度技术解析：从成本元素/金钱相关元素切入，详细分析低成本获得高价值的方法，目标受众为创业者，使用案例分析形式

6. **排序并输出结果**
   - 按照爆款概率从高到低排序所有热点
   - 参考 [references/output-format.md](references/output-format.md) 的格式规范输出
   - 确保输出包含：排序、热点标题、爆款元素列表、爆款概率、选题方向

### 可选分支
- 当 搜索结果不足10个时：说明搜索结果数量，并分析可能原因（关键词太窄、时效性要求等），建议用户调整关键词或时间范围
- 当 热点缺乏爆款元素时：说明该热点虽然热门但可能缺乏爆款潜力，建议从其他角度挖掘或等待后续发酵
- 当 热点涉及敏感内容时：提示用户注意内容合规性，建议谨慎选题或调整表达角度

## 资源索引
- 必要脚本：见 [scripts/search_hotspots.py](scripts/search_hotspots.py)（用途：联网搜索最近指定天数的热点信息，参数：keyword（赛道关键词）、days（天数，默认3）、count（返回数量，默认10））
- 领域参考：见 [references/爆款元素整理汇总.pdf](references/爆款元素整理汇总.pdf)（何时读取：步骤3分析爆款元素时，提供13大爆款元素的完整定义、判断标准、关键词和案例）
- 格式参考：见 [references/output-format.md](references/output-format.md)（何时读取：步骤6输出结果时，确保输出格式符合规范）

## 注意事项
- **爆款元素分析强制规范**：
  - 必须**严格**使用 [references/爆款元素整理汇总.pdf](references/爆款元素整理汇总.pdf) 中定义的13大爆款元素
  - **禁止**使用PDF中未定义的任何元素名称，例如：时效性、创新性、实用性、影响力、话题性、趋势性、情感共鸣、权威性等
  - 13大元素名称必须完全一致，包括"/"符号
  - 必须在分析前完整阅读PDF文档，提取所有爆款元素的定义、关键词和案例
  - 确保元素名称与PDF文档完全一致，包括标点符号
- **概率评估强制规范**：
  - 必须按照量化方法计算：高匹配度2分，中匹配度1分，组合效应3分
  - 禁止主观随意调整概率
  - 必须明确说明评分依据和计算过程
- **选题生成强制规范**：
  - 每个选题必须关联至少1个爆款元素
  - 元素名称必须与13大元素列表完全一致
  - 必须说明为什么这个元素支撑该选题的爆款潜力
- 仅在需要时读取参考，保持上下文简洁。
- 爆款概率预测应基于客观分析，避免过度乐观或保守。
- 选题方向建议应具体可执行，避免空泛描述。
- 充分利用智能体的语言理解、分析和创作能力，避免为简单任务编写脚本。
- 搜索脚本仅负责获取热点数据，分析、预测和选题建议由智能体完成。
