## 🧩 三、爆款标题句式公式（9大模板）

### 公式1：「悄悄上线」型
> 适用：大厂新功能首发，制造"抢先知道"的稀缺感

**句式**：`[品牌]悄悄上线了[产品/功能]，[意外结论/颠覆判断]`

**爆款案例**：
- 美团悄悄上线了生活Agent，懒人的春天真的要来了（10万+）
- 可灵AI深夜悄悄上线AI模特，他们这是要革电商的命（10万+）
- 腾讯悄悄上线了"Claude Code"，居然还支持微信登录（10万+）
- 腾讯悄悄出了个插件版"Cursor"，还跟微信小程序打通了（10万+）

**仿写模板**：
```
[品牌]悄悄[上线/开源/发布]了[产品名]，[出人意料的发现/颠覆性结论]
```

---

### 公式2：「一手实测/首发体验」型
> 适用：新品发布时抢占"第一波"，制造稀缺感和权威性

**句式**：`[一手实测/全网首发/一手体验] [产品名]，[感叹词/结论]`

**爆款案例**：
- 一手体验首款通用Agent产品Manus - 唯有惊叹（10万+，转发46440）
- 全网首发|一手内测字节OmniHuman-1，这就是当今最强的AI数字人（3.8万）
- 一手实测豆包手机助手，这就是当今手机Agent的天花板（9.3万）
- 一手实测Nano Banana Pro后，我总结了8种全新的超神玩法（9.4万）

**仿写模板**：
```
[一手实测/全网首发|一手内测] [产品名]，[这就是/唯有/此为][最高评价]
```

---

### 公式3：「反认知」型
> 适用：打破用户固有认知，制造认知冲突，引发好奇

**句式**：`[意想不到的主语/现象]，[反常识的结论]`

**爆款案例**：
- DeepSeek的提示词技巧，就是没有技巧（10万+）
- 你骂AI越狠，它反而越聪明？（4.6万）
- 倒反天罡，人类研究起模仿AI了（10万+）
- AI看不到的爱心，成了最棒的AI检测器（10万+）
- GPT4o生成的烂自拍，反而比我们更真实（10万+）

**仿写模板**：
```
[A]的[B]，居然是/就是/反而是[意想不到的C]
[你以为X的]，其实是/却是Y
```


**仿写模板**：
```

[时间词]+[我做了X]，我[好像/终于/才真的][发现了Y]
```

## ⚡ 六、标题生成实战步骤

### Step 1：确认选题类型
对应以下标题策略：
- **时效热点**→ 用公式3（悄悄上线）或公式2（一手实测）
- **方法干货**→ 用公式8（N分钟教你）或公式1（反认知）
- **行业趋势**→ 用公式6（宣判型）或骨架A
- **情感故事**→ 用公式9（情感共鸣）
- **批评观点**→ 用公式3（反认知）+ 骨架D

### Step 2：套入骨架
从4种骨架中选择1个，填入核心信息

### Step 3：加入S级词汇
在标题中植入至少1个S级词汇（刚刚/神级/天花板/居然/封神等）

### Step 4：检查3个维度
- ✅ **信息增量**：看完标题，用户能预判到有新知识/新发现吗？
- ✅ **情绪价值**：标题本身能引发好奇/震撼/共鸣吗？
- ✅ **标题张力**：去掉第一个词，标题还有吸引力吗？

### Step 5：生成20个，排序筛选
- 每种骨架各生成4-5个
- 覆盖不同情绪（好奇/震撼/共鸣/焦虑）
- 按"你会不会点进去看"主观感受排序

## 📋 七、爆款标题对比分析

### 同主题：好标题 vs 弱标题

| 弱标题（卡兹克实际低阅读文章） | 强标题（卡兹克实际高阅读文章） | 差距原因 |
|-------------------------------|-------------------------------|---------|
| 简单聊聊一下Google新发的视频模型Veo3.1 | 实测GPT-5：写作坠入谷底，编程一骑绝尘 | 后者有对比、有结论 |
| 大模型盲测竞技场到底是个啥？ | AI们数不清六根手指，这事没那么简单 | 后者有悬念和反认知 |
| 聊聊AI产品的护城河 | 腾讯元宝最终还是活成了微信的模样 | 后者有具体场景和画面感 |
| 0.6B 这种小模型到底有啥用？ | DeepSeek的提示词技巧，就是没有技巧 | 后者是完整的反认知句 |

**规律**：
1. 动词比名词强（发现/觉得/变成 > 是/有/叫）
2. 有画面感比抽象好（黑暗森林 > 某个概念）
3. 带"我"的视角比客观陈述点击率高
4. 结论句比问句稍强（除非问句本身已很强）

## 🔑 八、卡兹克标题的5个核心规律

### 规律1：「个人视角」让标题更可信
卡兹克80%的高阅读标题都带有第一人称视角：
- "我觉得"、"我发现"、"我总结"、"我潜伏进了"
- 比第三方陈述更有信任感

### 规律2：「对比」制造认知冲击
- 新 vs 旧（再见了Sora）
- 表面 vs 真相（DeepSeek的技巧就是没有技巧）
- 数据 vs 现实（烂自拍 vs 比我们更真实）

### 规律3：「惊叹词」放在标题末尾
- 唯有惊叹
- 封神
- 这是十年一遇的大机遇
- 懒人的春天真的要来了
- 此即未来

### 规律4：「冲突感」比平铺直叙点击率高3-5倍
- 有冲突：微信封了元宝（自己打自己）
- 无冲突：腾讯元宝功能介绍

### 规律5：「时效词+品牌+动作」是新闻类标题最稳的结构
```
[刚刚/深夜] + [品牌] + [动作词] + [产品/结果]
```
- 刚刚，智谱突然开源了GLM-4.5
- 深夜，美团悄悄上线了生活Agent
- 腾讯今日正式发布首个全栈AI IDE

| 简单聊聊X | 价值感低，像日记 | X的真相是/X让我发现了 |
| X到底是个啥 | 语气太弱，缺乏张力 | X，居然是/X原来可以这样 |
| 来了 | 过于泛化 | [品牌]刚刚上线了X，[意外结论] |
| 超详细X指南 | 八股味，缺个性 | 保姆级X教程，但我还是想劝你 |
| X的N个方法 | 太模板化 | 分享N个[修饰词]的X，[价值承诺] |
| 聊聊X | 同"简单聊聊" | X，让我看到了[惊人的Y] |
