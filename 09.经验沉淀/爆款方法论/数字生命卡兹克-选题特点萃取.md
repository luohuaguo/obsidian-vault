## 🎯 一、选题切入角度（12种模式）

### 1️⃣ 时效热点型（最快响应）
**特征**：第一时间跟进AI圈大事件，标题带"刚刚""深夜""正式上线"

**经典案例**：
- "OpenAI全新发布o1模型 - 我们正式迈入了下一个时代" (10万+)
- "刚刚，智谱突然开源了GLM-4.5，实现综合SOTA" (4.1万)
- "字节全新发布豆包AI视频模型 - 再见了Sora,你的时代过去了" (10万+)

**仿写公式**：
```
[时间词：刚刚/深夜/今天]，[品牌/公司] [动作：发布/上线/开源]了[产品名]。[个人第一反应+价值判断]
```

---

### 2️⃣ 首发实测型（独家体验）
**特征**：强调"一手实测""全网首发"，制造稀缺感

**经典案例**：
- "一手体验首款通用Agent产品Manus - 唯有惊叹" (10万+)
- "一手实测豆包手机助手,这就是当今手机Agent的天花板" (9.3万)
- "全网首发|一手内测字节OmniHuman-1,这就是当今最强的AI数字人" (3.8万)

**仿写公式**：
```
[一手实测/全网首发] [产品名/功能]，[个人体验感受 + 肯定判断]
```

---

### 3️⃣ 反认知型（颠覆常识）
**特征**：打破用户固有认知，制造认知冲突

**经典案例**：
- "你骂AI越狠,它反而越聪明?" (4.6万)
- "倒反天罡,人类研究起模仿AI了" (10万+)
- "DeepSeek的极致谄媚,正在摧毁我们的判断力" (4.6万)

**仿写公式**：
```
[反常识现象]，[意想不到的结论/真相]
```

---

### 4️⃣ 警示危机型（焦虑驱动）
**特征**：强调危机感、淘汰风险、紧迫感

**经典案例**：
- "Claude 3.7的编程能力太强了,强到会砸碎很多人的饭碗" (3.5万)
- "分析了1.8亿个岗位后,我发现应届生们好像被AI堵在了门外" (5.9万)
- "很多AI人还在自嗨,外贸人已经用AI卷翻天了" (3.7万)

**仿写公式**：
```
[数据/现象]后，我发现[危机/风险/真相]
```

---

### 5️⃣ 方法论拆解型（干货分享）
**特征**：分享具体方法、技巧、教程，数字量化价值

**经典案例**：
- "分享6个平时我最常用的Prompt心法" (8.7万)
- "一手实测Nano Banana Pro后,我总结了8种全新的超神玩法" (9.4万)
- "分享10个我最常用的DeepResearch提示词模板和用法" (3.3万)

**仿写公式**：
```
分享[N]个[形容词]的[主题]方法/技巧/玩法
```

---

### 6️⃣ 人物故事型（专访/案例）
**特征**：通过具体人物故事切入，增强可信度和代入感

**经典案例**：
- "专访刘世奇 - 他用AI设计丑拖鞋,一年卖了3000万" (7.3万)
- "专访"Prompt之神"李继刚 - 我想用20年时间,给世界留一句话" (6.2万)
- "专访阿里国际站总裁张阔:给AI时代生意人的9条启示" (2.6万)

**仿写公式**：
```
专访[人物] - [人物故事/成就/金句]
```

---

### 7️⃣ 行业趋势型（高维认知）
**特征**：从宏观角度分析行业趋势，满足用户"看门道"需求

**经典案例**：
- "一文带你通俗易懂的看懂'人工智能+',这是十年一遇的大机遇" (10万+)
- "唐杰、杨植麟、姚顺雨、林俊旸罕见同台分享,这3个小时的信息密度实在太高了" (9.5万)
- "一文看懂2025 Google I/O开发者大会" (2.6万)

**仿写公式**：
```
一文[看懂/带你了解] [复杂话题]，这是[价值判断]
```

---

### 8️⃣ 趣味玩梗型（娱乐传播）
**特征**：结合热点梗、二次元、娱乐元素，降低认知门槛

**经典案例**：
- "Grok火爆全球,靠的居然是一个二次元金发美少女" (3.1万)
- "GPT4o生成的烂自拍,反而比我们更真实" (10万+)
- "笑疯了,AI换装大法让奥特曼都性感起来了" (0.9万)

**仿写公式**：
```
[趣味现象/梗]，[意想不到的关联]
```

---

### 9️⃣ 情感共鸣型（走心内容）
**特征**：触及情感、人性、生死、亲情等话题

**经典案例**：
- "他用Luma和Suno复活了逝去11年的爱人,给我看破防了" (10万+)
- ""我要去世了,豆包。"" (10万+)
- "当身边亲人离去的那一刻,我才真的懂了数字生命的意义" (1.2万)

**仿写公式**：
```
[感人故事/场景]，给我看破防了/我懂了[深刻道理]
```

---

### 🔟 批判反思型（价值观输出）
**特征**：对AI乱象进行批判，展现独立思考

**经典案例**：

- "AI领域的全面战争,从AI爬虫毁灭互联网开始" (3.1万)
- "对于那些出来卖的DeepSeek课程,我有些话想说" (3.9万)

**仿写公式**：
```
[批判对象]，[犀利评价/问题本质]
```

## 📝 二、标题句式特征

|------|------|---------|
| [数据/现象]后，我发现... | 分析了1.8亿个岗位后,我发现... | 高 |
| 一文[看懂/带你]... | 一文带你通俗易懂的看懂... | 高 |

## 🔑 三、选题判断标准

### 卡兹克选题的5个核心标准

1. **信息增量** - 用户看完能获得新知识/新认知
2. **情绪价值** - 能引发惊讶、好奇、焦虑、共鸣
3. **时效性** - 越快越好，"刚刚"比"昨天"好
4. **个人视角** - 必须有"我"的实测/体验/观点
5. **标题张力** - 标题本身就能引发点击欲望

### 卡兹克选题优先级

| 优先级 | 选题类型 | 理由 |
|--------|---------|------|
| S级 | 重大AI事件首发实测 | 稀缺性+时效性+权威性 |
| A级 | 爆款工具深度教程 | 实用性+收藏价值 |
| A级 | 行业趋势深度解读 | 高维认知+转发价值 |
| B级 | 人物故事/专访 | 情感共鸣+可信度 |
| B级 | 反认知/批判观点 | 立场鲜明+讨论价值 |
| C级 | 趣味玩梗/日常分享 | 传播性强但深度有限 |

## 🎬 四、开篇模式（从TOP5案例提取）

### 模式1：时间锚点切入
```
[时间词]，[发生了什么/我做了什么]。[为什么值得关注]。

例："今天晚上,朋友圈被一份文件刷屏了。《关于深入实施'人工智能+'行动的意见》。人工智能+,等了好久,他来了。"
```

### 模式2：事件引入+悬念
```
[热点事件背景]。[关键人物/细节]一定是[看到了什么/做了什么]，才[做出反常举动]。

例："最近OpenAI的宫斗,OpenAI首席科学家Ilya一脚将奥特曼踢出局。在吃瓜之余,引发了一群大佬们关于AI安全的辩论。马斯克说的非常准确,Ilya一定是是看到了什么,才如此激进。"
```

### 模式3：亲身经历切入
```
人在[地点/场景]，[有感而发/趁热打铁]。[发生了什么]。

例："人在从杭州回北京的路上,想趁着脑子里的记忆还火热的时候,写下一些感悟。"
```

### 模式4：提问引入
```
[提出一个反常识/有趣的问题]？[简要铺垫]。[引出本文主题]。

例："四万字长文带你了解为什么超级大佬们都如此警惕人工智能？Ilya到底在害怕什么？"
```

## 📐 五、内容结构模板

### 教知识型结构
```
【开篇】痛点/热点切入 → 为什么这个话题重要
【主体】
  - 工具/方法介绍
  - 具体操作步骤（带截图）
  - 多个应用场景
【结尾】总结+行动号召
```

### 聊观点型结构
```
【开篇】事件/现象引入 → 我的立场
【主体】
  - 观点1 + 论据
  - 观点2 + 论据
  - 观点3 + 论据
【结尾】金句收尾 + 升华
```

### 讲故事型结构
```
【开篇】人物/场景引入
【主体】
  - 背景故事
  - 关键转折
  - 成功/失败结果
【结尾】启发/金句
```

## 💡 六、选题灵感来源

### 卡兹克日常选题来源

1. **海外AI动态**：Twitter/X、Reddit、Product Hunt
2. **国内AI发布**：各大厂公众号、官方渠道
3. **用户反馈**：评论区、私聊、社群问题
4. **行业活动**：发布会、大会、展会现场
5. **个人使用**：日常用AI过程中的发现

### 选题验证清单

- [ ] 是否有时效性？（越新越好）
- [ ] 是否有信息增量？（用户能学到东西）
- [ ] 是否有情绪价值？（能引发反应）
- [ ] 是否有个人视角？（有"我"的参与）
- [ ] 标题是否有张力？（能否引发点击）

## 🔄 七、如何应用这份萃取

### 选题阶段
1. 从热点链接中提取核心信息
2. 对照12种切入角度，选择最合适的2-3种
3. 用仿写公式生成标题候选
4. 用选题验证清单打分

### 写作阶段
1. 根据内容类型选择结构模板
2. 参考开篇模式，选择最合适的切入方式
3. 注入个人视角和实测体验
4. 使用高频词汇增强标题张力

### 打磨阶段
1. 检查是否有信息增量
2. 检查是否有情绪价值
3. 检查标题是否符合句式特征
4. 检查结尾是否有金句
